# DSA0409-Fundamentals-of-Data-Science

Welcome to the **DSA0409 Fundamentals of Data Science** repository! This repository contains a comprehensive collection of Python exercises and experiments covering essential data science concepts, statistical analysis, and machine learning techniques.

## üìö Course Overview

This course (December 2025) introduces fundamental concepts in data science through hands-on programming exercises. The curriculum covers:

- **Data Manipulation**: NumPy arrays, Pandas DataFrames
- **Data Visualization**: Matplotlib, plotting techniques
- **Statistical Analysis**: Descriptive statistics, hypothesis testing, confidence intervals
- **Machine Learning**: Classification, regression, clustering algorithms
- **Text Processing**: Natural language processing, word frequency analysis

## üìÇ Repository Structure

The repository contains 40 exercises organized into two main categories:

### Python Scripts (`.py` files)
- **ex1.py - ex6.py**: Basic exercises on NumPy operations, array manipulations, and fundamental data analysis
- **Exp_7.py - Exp_40.py**: Advanced experiments covering various data science topics

### Jupyter Notebooks (`.ipynb` files)
- **ex16.ipynb - ex23.ipynb**: Interactive notebooks with detailed scenarios and comprehensive solutions

## üî¨ Topics Covered

### 1. **NumPy Fundamentals**
- Array operations and manipulations
- Statistical calculations (mean, median, standard deviation)
- Matrix operations
- Example: `ex1.py` - Student scores analysis across subjects

### 2. **Pandas Data Analysis**
- DataFrame creation and manipulation
- Data aggregation and grouping
- Time series analysis
- Example: `Exp_7.py` - Customer order analysis and date handling

### 3. **Data Visualization**
- Line plots, bar charts, scatter plots
- Box plots for statistical distribution
- Q-Q plots for normality testing
- Example: `Exp_10.py` - Monthly sales visualization

### 4. **Statistical Analysis**
- Descriptive statistics
- Frequency distributions
- Confidence intervals
- Hypothesis testing (t-tests, A/B testing)
- Examples: `ex18.ipynb` - Body fat analysis, `ex19.ipynb` - Clinical trial confidence intervals

### 5. **Machine Learning**

#### Classification
- Decision Tree Classifier
- Iris species classification
- Example: `Exp_25.py` - Flower species prediction

#### Regression
- Decision Tree Regressor
- Price prediction with decision path analysis
- Example: `Exp_30.py` - Used car price prediction

#### Clustering
- K-Means clustering implementation
- Customer segmentation
- Example: `Exp_40.py` - Customer segmentation based on spending patterns

### 6. **Text Processing & NLP**
- Word frequency analysis
- Text preprocessing (removing punctuation, stop words)
- Customer feedback analysis
- Example: `ex16.ipynb`, `ex17.ipynb` - Word frequency distribution in customer reviews

## üöÄ Getting Started

### Prerequisites

Ensure you have Python 3.x installed along with the following libraries:

```bash
pip install numpy pandas matplotlib scipy scikit-learn nltk jupyter
```

### Installation

1. Clone the repository:
```bash
git clone https://github.com/MR-WHOAMEYE/DSA0409-Fundamentals-of-Data-Science.git
cd DSA0409-Fundamentals-of-Data-Science
```

2. Install required dependencies:
```bash
pip install numpy pandas matplotlib scipy scikit-learn nltk jupyter
```

3. For NLTK resources (required for text processing exercises):
```python
import nltk
nltk.download('stopwords')
```

## üìñ Usage

### Running Python Scripts

Execute any Python script directly:
```bash
python ex1.py
python Exp_10.py
```

Some scripts are interactive and will prompt for user input (e.g., `Exp_25.py`, `Exp_30.py`).

### Running Jupyter Notebooks

1. Launch Jupyter Notebook:
```bash
jupyter notebook
```

2. Navigate to the notebook file (e.g., `ex16.ipynb`) and open it in your browser.

3. Run cells sequentially to see the analysis and results.
